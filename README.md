## Overview

This repository contains the latest releases of the files required to collect data from applications that need patching.

Applications can be patched directly using **Mover**. The files provided here are intended for **manual installation only**.

## When to Use These Files

You may need to perform a manual installation in the following cases:

- Troubleshooting or tracking a specific issue  
- Running Mover on a different machine than the application, requiring configuration of the Mover machine’s IP address (via UDP)  
- Customizing the data sent to Mover (supported in some patches)

## Notes

- Most files are compressed and must be extracted before use
